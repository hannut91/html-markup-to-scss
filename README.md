# html-markup-to-scss

## How to use

### Installation
> npm install

### Run
> node app.js

1. Copy html string to clipboard
2. Please enter in terminal
3. Paste scss string to .scss file

### Example
input :
```html
<div class="test-wrap"> 
  <div class="test">
  </div>
</div>
```
output : 
```scss
.test-wrap {
  .test {
  }
}
```
